# Manejo de archivos

* Crear una carpeta 
```bash
mkdir name_file
```
* Crear un archivo
```bash
tocuh text.txt 
```
* Eliminar archivos
```bash
rm text.txt
```
* Eliminar carpetas
```bash
rmdir name_file
```

* Borrar una carpeta con archivos
```bash
rm -rf name_file
```
